# ChatBotWebApiApplication - Chatbot based on AI 

This project represents my first venture into web development, coming from an 8-year tenure at "Twoteam" Company, where I was involved in developing desktop software using WPF. With no professional experience in web technologies, embarking on this journey was both thrilling and challenging. This entire project was self-taught, showcasing my commitment to learning and expanding my skill set.

Throughout this development process, I've learned to utilize various tools and technologies to bring this project to life. Key achievements include:

Initiating the project from scratch: Starting a project without prior experience in web development was a significant leap for me, and it has been an exciting learning curve.

Self-learning: Diving into new territories, I familiarized myself with the nuances of web development, understanding the intricacies involved in creating a web application.

Integrating with OpenAI's API: One of the highlights of this project was successfully integrating with OpenAI's API, which allowed me to implement an interactive chat feature.

Exploring Swagger: Learning to use Swagger was instrumental in testing the application, providing a user-friendly interface to interact with the web services.

Installing Docker: The installation and utilization of Docker marked a crucial step in ensuring the application's scalability and deployment efficiency.

As I continue to expand this project, including the development of the client-side, I look forward to further enhancing my web development skills and exploring more advanced features and technologies.
